`sanity-editor` is a Nuxt module that exposes a TipTap-based rich-text editor using **Portable Text blocks** as the model.

At a high level:

- Your app stores rich text as `PortableTextBlock[]`
- The editor shows and edits that content as TipTap/ProseMirror JSON
- Conversion helpers keep the mapping between TipTap JSON and Portable Text blocks
