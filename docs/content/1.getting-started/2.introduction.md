`sanity-editor` is an **experimental Nuxt module** for editing [Portable Text](https://portabletext.org/) in the browser.

It provides a TipTap (ProseMirror JSON) editor and bridges it to/from Portable Text blocks (`PortableTextBlock[]`).

## Data model (PortableTextBlock[])

At runtime, `sanity-editor` works with arrays of Portable Text blocks shaped like:

```ts
type PortableTextBlock = {
  _type: string
  _key: string
  // other fields depend on your schema
}
```

You connect the editor to your data via `v-model` on `<PortableTextEditor />`.

## Supported built-in blocks & marks

Out of the box, the module includes:

- Block styles: `normal`, `h1` .. `h6`, `blockquote`
- Lists: `bullet` and `number`
- Inline marks: `strong`, `em`, `strike-through`, `code`
- `link` annotation (`href`)
- Block object: `horizontal-rule` (mapped to TipTap `horizontalRule`)

To support other `_type` values (like Sanity `task` or custom objects), add **Portable Text block extensions**.

## What you install

The module itself registers:

- `<PortableTextEditor />` (client component)
- `usePortableTextEditor()` composable
- Conversion helpers for TipTap JSON <-> Portable Text

You also need to have `nuxt-tiptap-editor` available in your app (as a peer dependency).
