**sanity-editor** is an **experimental** [Nuxt 4](https://nuxt.com) module that lets you edit **[Portable Text](https://portabletext.org/)** in the browser using **[TipTap](https://tiptap.dev/)**. It pairs with [`nuxt-tiptap-editor`](https://github.com/modbender/nuxt-tiptap-editor) (registered automatically).

## What is Portable Text?

Portable Text is a JSON-based rich text format used heavily in the [Sanity](https://www.sanity.io/) ecosystem: blocks, spans, marks, and custom block types. It is **not HTML**—it is structured data you can validate, query, and render on any platform.

## Why not HTML?

Many WYSIWYG editors round-trip through HTML. That works for generic pages, but Portable Text workflows expect a **stable block model** aligned with `@portabletext/schema`. This module bridges **TipTap / ProseMirror JSON** and Portable Text so your `v-model` stays `PortableTextBlock[]` end to end.

## Experimental status

The API, schema coverage, and real-world Sanity workflows are still evolving. Please share feedback via [GitHub Issues](https://github.com/isakgustavsen/sanity-editor/issues) or [Discussions](https://github.com/isakgustavsen/sanity-editor/discussions).

## Requirements

- **Nuxt** `^4`
- **`nuxt-tiptap-editor`** `^3` (install alongside `sanity-editor`; the module registers it)
