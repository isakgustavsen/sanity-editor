## Setup

1. **Install the module**

   Add **`sanity-editor`** only. The module declares **`nuxt-tiptap-editor`** as a **`moduleDependency`** (version `>=3.2.0`), so Nuxt installs and wires it when your app prepares—no separate package line required.

   ::code-group

   ```bash [pnpm]
   pnpm add sanity-editor
   ```

   ```bash [npm]
   npm install sanity-editor
   ```

   ```bash [yarn]
   yarn add sanity-editor
   ```

   ::

2. **Enable the module in Nuxt**

   ```ts [nuxt.config.ts]
   export default defineNuxtConfig({
     modules: ['sanity-editor'],
   })
   ```

3. **Use module options when needed**

   Module options are read from the `sanityEditor` key in `nuxt.config.ts`.

   ```ts [nuxt.config.ts]
   export default defineNuxtConfig({
     modules: ['sanity-editor'],
     sanityEditor: {
       // optional module options
     },
   })
   ```

`sanity-editor` runs **`installModule('nuxt-tiptap-editor')`** during setup, so you do not add `nuxt-tiptap-editor` to `modules` yourself.

## Type your model value

Use `PortableTextBlock` from `@portabletext/types` for your editor state.

You're ready to render your first editor instance.
