## Install packages

Add **`sanity-editor`** and the peer **`nuxt-tiptap-editor`**:

```bash [pnpm]
pnpm add sanity-editor nuxt-tiptap-editor
```

```bash [npm]
npm install sanity-editor nuxt-tiptap-editor
```

```bash [yarn]
yarn add sanity-editor nuxt-tiptap-editor
```

## Register the module

In `nuxt.config.ts`, add **`sanity-editor`**:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['sanity-editor'],
})
```

You do **not** need to add `nuxt-tiptap-editor` to `modules` yourself—the **sanity-editor** module registers it for you.

If you pass module options in `nuxt.config`, use the **`sanityEditor`** key (the module’s `configKey`).

## Types

Use `PortableTextBlock` from **`@portabletext/types`** in your components when typing `v-model` data.
