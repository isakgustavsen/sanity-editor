---
title: Installation
description: Install and register the sanity-editor Nuxt module.
---

## Requirements

- Nuxt 4
- `nuxt-tiptap-editor` (this module installs it as a dependency)

## Install

```bash
pnpm add sanity-editor nuxt-tiptap-editor
# or: npm install sanity-editor nuxt-tiptap-editor
```

## Register in `nuxt.config.ts`

```ts
export default defineNuxtConfig({
  modules: ['sanity-editor'],
})
```

## Quick sanity check

Open the playground at `playground/` and verify the editor renders:

```bash
pnpm run dev:prepare
pnpm dev
```
