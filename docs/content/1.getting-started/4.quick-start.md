This quick start uses the main `<PortableTextEditor />` component.

## 1) Create your Portable Text state

In Sanity, your editor typically works with an array of Portable Text blocks (from `@portabletext/types`):

```ts
import type { PortableTextBlock } from '@portabletext/types'

const blocks = ref<PortableTextBlock[]>([])
```

## 2) Use the component

```vue
<script setup lang="ts">
import type { PortableTextBlock } from '@portabletext/types'

const blocks = ref<PortableTextBlock[]>([])
</script>

<template>
  <PortableTextEditor v-model="blocks" />
</template>
```

## 3) Add a toolbar (optional)

`<PortableTextEditor />` exposes an optional `toolbar` slot.

```vue
<PortableTextEditor v-model="blocks">
  <template #toolbar="{ editor }">
    <button type="button" @click="editor.chain().focus().toggleBold().run()">
      Bold
    </button>
    <button type="button" @click="editor.chain().focus().toggleItalic().run()">
      Italic
    </button>
  </template>
</PortableTextEditor>
```

If you don’t provide a slot, the component renders a minimal default toolbar.
