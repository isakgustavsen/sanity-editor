Use the auto-imported **`SanityEditor`** component with a ref typed as **`PortableTextBlock[]`**:

```vue [app.vue]
<script setup lang="ts">
import type { PortableTextBlock } from '@portabletext/types'

const content = ref<PortableTextBlock[]>([])
</script>

<template>
  <SanityEditor v-model="content" />
</template>
```

For SSR, keep the editor in a **client-only** boundary if you prefer not to ship TipTap on the server:

```vue
<script setup lang="ts">
import type { PortableTextBlock } from '@portabletext/types'

const content = ref<PortableTextBlock[]>([])
</script>

<template>
  <ClientOnly>
    <SanityEditor v-model="content" />
  </ClientOnly>
</template>
```

The module exposes **`SanityEditor`**, **`useSanityEditor`**, conversion helpers, and default schema exports—see the module’s `addImports` in the package source.
