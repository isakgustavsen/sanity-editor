## Minimal example

Use the auto-imported **`PortableTextEditor`** component with a ref typed as **`PortableTextBlock[]`**:

```vue
<script setup lang="ts">
import type { PortableTextBlock } from '@portabletext/types'

const content = ref<PortableTextBlock[]>([])
</script>

<template>
  <PortableTextEditor v-model="content" />
</template>
```

## Reading the data

`content` is always an array of Portable Text blocks. You can log it, send it to an API, or pass it to [`@portabletext/to-html`](https://www.npmjs.com/package/@portabletext/to-html) / your own serializers.

```vue
<script setup lang="ts">
import type { PortableTextBlock } from '@portabletext/types'

const content = ref<PortableTextBlock[]>([])
</script>

<template>
  <div>
    <PortableTextEditor v-model="content" />
    <pre class="mt-4 text-xs overflow-auto">{{ JSON.stringify(content, null, 2) }}</pre>
  </div>
</template>
```

## Auto-imports

The module exposes **`PortableTextEditor`**, **`usePortableTextEditor`**, conversion helpers, and default schema exports—see the module’s `addImports` in the package source.
