## Overview

`SanityEditor` is a client component that wires **`useSanityEditor`** to **`TiptapEditorContent`** and provides an optional default toolbar.

## Slots

| Slot | Scope | Description |
| --- | --- | --- |
| `toolbar` | `{ editor: Editor \| null }` | Replaces the default Bold / Italic / H2 / List toolbar. Only rendered when `editor` is truthy. |

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `v-model` | `PortableTextBlock[]` | `[]` | Blocks to edit (Portable Text shape from `@portabletext/types`). |
| `schema` | `Schema` | тАФ | Compiled `@portabletext/schema` schema. |
| `extensions` | `AnyExtension[]` | тАФ | Additional TipTap extensions. |
| `blockExtensions` | `SanityEditorBlockExtension[]` | тАФ | Custom TipTap тЖФ block mappings. |
| `editorProps` | `Record<string, unknown>` | тАФ | Extra TipTap editor options (see composable). |
| `keyGenerator` | `() => string` | тАФ | Key generator for new `_key` values in conversion. |

Types are imported from **`@portabletext/types`**, **`@portabletext/schema`**, and **`@tiptap/core`** (`AnyExtension`).
