## Overview

`PortableTextEditor` is a client component that wires **`usePortableTextEditor`** to **`TiptapEditorContent`** and provides an optional default toolbar.

## Slots

| Slot | Scope | Description |
| --- | --- | --- |
| `toolbar` | `{ editor: Editor \| null }` | Replaces the default Bold / Italic / H2 / List toolbar. Only rendered when `editor` is truthy. |

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `v-model` | `PortableTextBlock[]` | `[]` | Portable Text blocks to edit. |
| `schema` | `Schema` | — | Compiled `@portabletext/schema` schema. |
| `extensions` | `AnyExtension[]` | — | Additional TipTap extensions. |
| `portableTextBlockExtensions` | `PortableTextBlockExtension[]` | — | Custom TipTap ↔ PT block mappings. |
| `editorProps` | `Record<string, unknown>` | — | Extra TipTap editor options (see composable). |
| `keyGenerator` | `() => string` | — | Key generator for new `_key` values in conversion. |

Types are imported from **`@portabletext/types`**, **`@portabletext/schema`**, and **`@tiptap/core`** (`AnyExtension`).
