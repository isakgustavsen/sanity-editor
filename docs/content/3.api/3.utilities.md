## `createPortableTextContext`

```ts
function createPortableTextContext(
  options?: CreatePortableTextContextOptions,
): PortableTextTransformContext

interface CreatePortableTextContextOptions {
  schema?: Schema
  keyGenerator?: () => string
  portableTextBlockExtensions?: PortableTextBlockExtension[]
}
```

Returns a **`PortableTextTransformContext`** for **`portableTextToTipTapJson`** and **`prosemirrorJsonToPortableText`**, defaulting **`schema`** to **`defaultCompiledPortableTextSchema`**. Use this when you integrate your own editor surface (for example Nuxt UI **`UEditor`**) instead of **`usePortableTextEditor`**.

## Conversion functions

### `prosemirrorJsonToPortableText`

```ts
function prosemirrorJsonToPortableText(
  docJson: unknown,
  ctx: PortableTextTransformContext,
): PortableTextBlock[]
```

Converts a TipTap **`editor.getJSON()`** document (root `type: 'doc'`) into an array of **`PortableTextBlock`**.

### `portableTextToTipTapJson`

```ts
function portableTextToTipTapJson(
  blocks: PortableTextBlock[],
  ctx: PortableTextTransformContext,
): TiptapJSONDoc
```

Converts Portable Text blocks into TipTap/ProseMirror JSON suitable for **`setContent`**.

### `PortableTextTransformContext`

| Field | Type | Description |
| --- | --- | --- |
| `schema` | `Schema` | Required compiled schema for validation-style transforms. |
| `keyGenerator` | `() => string` | Optional; defaults to random base36 keys. |
| `portableTextBlockExtensions` | `PortableTextBlockExtension[]` | Optional custom block mappings. |

## JSON types

Exported for typing:

- **`TiptapJSONDoc`** — `{ type: 'doc', content?: TiptapJSONNode[] }`
- **`TiptapJSONNode`**, **`TiptapJSONMark`**

## Default schema exports

| Export | Description |
| --- | --- |
| `defaultPortableTextSchemaDefinition` | `defineSchema({ ... })` from `@portabletext/schema`. |
| `defaultCompiledPortableTextSchema` | `compileSchema(defaultPortableTextSchemaDefinition)` — use as default **`schema`** in context. |

Use these when you call the conversion helpers outside the editor. Prefer **`createPortableTextContext()`** over assembling the context object by hand.
